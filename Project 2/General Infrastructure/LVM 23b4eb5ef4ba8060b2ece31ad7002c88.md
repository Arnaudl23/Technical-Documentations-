# LVM

```arduino
lsblk (voir les disques)

sudo fdisk /dev/sdc (Partitionner un nouveau disque)
Étapes dans fdisk :

n → Nouvelle partition

p → Primaire

1 → Numéro de la partition

Entrée → Début par défaut

Entrée → Fin par défaut

t → Changer le type

8e → Type Linux LVM

w → Écrire et quitter

sudo pvcreate /dev/sdc1 (sudo pvcreate /dev/sdc1)

sudo vgcreate wp-vg /dev/sdc1 (Créer le Volume Group (VG))

sudo lvcreate -L 10G -n wp-lv wp-vg (Créer le Logical Volume (LV))

sudo mkfs.ext4 /dev/wp-vg/wp-lv (Formater en EXT4)

sudo mkdir -p /var/www/html/wordpress
sudo mount /dev/wp-vg/wp-lv /var/www/html/wordpress (monter le dossier)

sudo blkid /dev/wp-vg/wp-lv (Récupérer l’UUID)

UUID=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx /var/www/html/wordpress ext4 defaults 0 2 (Ajouter dans /etc/fstab)

```
